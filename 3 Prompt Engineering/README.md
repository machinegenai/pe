# BJ's AI Champions — Prompt Engineering Course

The full Prompt Engineering training program for BJ's AI Champions, organized by level.

## What's in this folder

- **0 - Crash Course (Start Here)/** — `Prompt_Engineering_All-in-One` (20 slides). The whole journey in one session: every technique, condensed, with real before/after examples. Best for a kickoff, a refresher, or anyone short on time.
- **1 - Beginner/** — *From Asking to Composing.* CO-STAR + the Refine loop. Includes `Beginner_Facilitator_Demo_Script.md` (the live Claude demos).
- **2 - Intermediate/** — *From Composing to Directing.* Show-it-examples, reason-first, structure a long prompt, work from your own documents, diagnose & refine.
- **3 - Advanced/** — *From Directing to Designing.* Chain the work, reusable templates, context engineering, evaluate outputs, guardrails.
- **Planning/** — `Prompt_Engineering_Curriculum_Map.xlsx`. The full curriculum map and coverage dashboard.

## Each level has two files

- **`.pptx`** — the facilitator master. Speaker notes on every slide (facilitation cues + the technical terms kept off the slides).
- **`.pdf`** — slides only, for SharePoint self-read.

## The journey

**Asking → Composing → Directing → Designing.** Beginner teaches Composing, Intermediate teaches Directing, Advanced teaches Designing. The crash course walks all four in one pass. A recurring roadmap slide shows where each level sits.

## One running example

Intermediate, Advanced, and the crash course all work one real batch of ~30 member-feedback comments (checkout waits, produce, fresh food, the app, the gas pump). Every quote and count on every slide traces back to it. Beginner uses a single member complaint (a $60 auto-renewal dispute).

## Before you present externally

All material is illustrative — footered *"not an official BJ's publication."* The policy details used in examples (renewal/refund windows, data-safety wording, responsible-use rules) are placeholders — swap them for BJ's real policy before using any output with a member.
